# 🚀 CoderHN Core V1 - Astro Starter Template

Esta es la plantilla base oficial de **CoderHN** para proyectos web modernos, rápidos y elegantes. Construida con **Astro** y **Tailwind CSS**, optimizada para SEO y rendimiento.

## ✨ Características Principales

- 🌑 **Dark Mode Ready:** Configurado con la clase de Tailwind y script anti-destello (FOUC).
- 🎨 **Diseño Moderno:** Header con efecto *Glassmorphism* y componentes minimalistas.
- 📱 **Totalmente Responsivo:** Adaptado para móviles, tablets y escritorio.
- ⚡ **Performance:** 100/100 en Lighthouse gracias a la arquitectura de Astro.
- 🛠️ **Fácil Configuración:** Variables de color CSS centralizadas.

## 🛠️ Tecnologías

- [Astro](https://astro.build/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Lucide Icons](https://lucide.dev/)
